Command Line and Git Cheat Sheet

Misc

pwd print working directory - tells you where you are
ls find out what is in directory
cd change directory
mkdir make directory
touch create a file
echo displays a string
cp copy a file
mv move a file
rm delete a file
ctrl + c cancel
tab finishes the word for you

Git

git init initiate git
git add --all adds changes to git
git commit -m "Message" commits the changes with message
git status checks status
git push pushes to github
git pull pulls the code down / updates the file
git clone pulls down the code - clones it
